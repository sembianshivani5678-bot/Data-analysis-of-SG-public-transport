# Data-analysis-of-SG-public-transport
Python project to analyse transport distruptions in MRT lines and suggestion of alternative travel routes

# Tools used 
Python
Pandas

# Process
Collected MRT timing data from official SG transport platforms
Sorted out the Data into tables to analyse it
Analysed the duration of delays during breakdowns of trains
The data I included in my project is live data from the government which would help me analyse the whether any distruptions in the MRT lines would cause delays to my travel times

# Important Findings
Major breakdowns caused delays up to 5 hours while minor delays are within 30 minutes
In 2025, North-East Line had the most train breakdowns, while in 2026, NEL line has topped the chart for being the most efficient line 
Delays are also caused by construction happening to upgrade the trails 

# Suggestion of efficient travel
I collected data to analyse which line is the most efficient to travel at specific periods of the day 
The government has also arranged shuttle buses at MRT stations with high crowd frequency to help reduce significant delays 

# Conclusion
This project allows me to keep updated of the disruptions in the MRT lines and helps suggest better travel routes so that I do not have delays when travelling. 
The start of this project started when I experienced my first train disruption when going to school in Secondary 4. This disruption caused me to be late to school for the first time and it had me thinking why there was a disruption as I used to think that Singapore's transport systems had no errors in them. This led me to research more about the reasons of the disruptions. Through which I found data of many disruptions, minor to major, in the MRT lines in the past which I never knew about. This made me realise this data showed me trends which could be analysed to avoid further delays in the future. I started collecting data and slowly started writing codes to analyse the data systematically. Recently, I uploaded it to GitHub to keeo track of my project.
